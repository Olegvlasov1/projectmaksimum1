Виртуальное окружение -- это папка которая хранит библиотеки
py -m venv venv -- создание ВО
venv/Scripts/activate  -- активация ВО
открыть PowerShell от им админа
Ввести Set-ExecutionPolicy RemoteSigned -- A
pip install django
py manage.py runserver
django-admin startproject core  -- запуск сервера
http://127.0.0.1:8000/ -- ссылка локального сервера
web-app -- отдельная часть которая отвечает за определенный функционал
    например для вк:
        - вход, выход, регистрация
        - чат
        - главная страница и т.д.
py manage.py startapp <name> -- создание веб приложения
приповторном входе нужно опять активировать ВО
при создании нового веб приложения его нужно указать в настройках
